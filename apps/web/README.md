## Staff Onboarding

Staff can submit onboarding details and weekly timesheets at `/staff`.

Office review is available at `/admin/staff`.

Required deployment step after schema changes:

```bash
npx prisma db push
```

Optional email routing:

```bash
STAFF_EMAIL_TO=enquiries@almagroup.com.au
```

Xero Payroll connection still needs a Xero developer app and OAuth credentials before live payroll sync is enabled. For Australian payroll, the integration will need the Payroll AU Employees API, a `Xero-Tenant-Id`, and employee write scope such as `payroll.employees`. Keep tax file numbers and bank details out of the public staff form until that secure Xero payroll connection is configured.
